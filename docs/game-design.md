# 水果消消乐游戏设计文档

## 1. 游戏概述
水果消消乐是一款类似于“莲手”的方格消除游戏，玩家通过点击相同水果进行重组，消除满足条件的连接以获得分数。

## 2. 核心游戏逻辑

### 2.1 游戏卡片设计
- **水果种类**：苹果、香蕉、西瓜、葡萄、橙子、荆棘猫等
- **特殊格式**：万能快、小火炮、钟面等
- **游戏区域**：8x8的方格间隙

### 2.2 消除机制
1. **消除条件**：
   - 同一行或同一列上有3个或更多相同水果
   - 可以是任意方向的连续水果

2. **消除效果**：
   - 消除后水果下落，上面的水果向下移动
   - 上方补充新的随机水果
   - 消除时摇笑动画效果

3. **连锁机制**：
   - 消除后立即检测是否有新的消除组合
   - 有则续续消除，直到没有新的消除为止

### 2.3 分数系统
- **基础分数**：消除3个获得10分，4个20分，5个30分
- **连锁加成**：按连续消除次数叠加分数
- **特殊助手**：使用后获得大量分数
- **分数标志**：每当前最高分数

### 2.4 特殊功能
1. **物理效果**：
   - 万能快：消除周围所有水果
   - 小火炮：消除直线上的水果
   - 钟面：随机消除水果

2. **游戏环境**：
   - 时间限制：60秒内完成游戏
   - 步数限制：最多20步
   - 目标分数：实时显示还需要多少分

3. **游戏模式**：
   - 简单模式：无限制，目标是得到最高分
   - 挑战模式：有时间/步数限制，达到指定分数

## 3. 游戏状态机

### 3.1 主要状态
- **菜单状态**：开始界面、设置选择、成绩查看
- **游戏中状态**：正常游戏、计时中、消除中
- **游戏结束状态**：胜利、失败、屏幕反输

### 3.2 状态转换
```菜单 -> 游戏中 <-> 游戏结束
     ^              |
     |______________|
        重新开始
```

## 4. 效果声音
- **操作音**：点击、消除、摇笑
- **背景音乐**：轻快意悦的铃铛音乐
- **特效音**：连锁、特殊助手使用

## 5. 持续性设计
- **成就系统**：每天登录、特殊元素解锁
- **排行档**：全局最佳战绩
- **关卡系统**：不同难度的关卡选择

## 6. 技术标准
- **前端框架**：React + TypeScript
- **状态管理**：Redux或 Zustand
- **动画库**：GSAP或 CSS3动画
- **声音引擎**：Howler.js
- **存储**：LocalStorage + IndexedDB

## 7. 开发计划
1. **第一阶段**：基础游戏核心逻辑
2. **第二阶段**：UI界面和交互
3. **第三阶段**：动画效果和声音
4. **第四阶段**：游戏环境和模式
5. **第五阶段**：持续性功能和优化
