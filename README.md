# Fruit Match Game

一个精彩的水果消消乐游戏，让你体验连消的快感！

## 游戏特色

- ✨ 精美的水果全方位符
- ⚡ 灾厄符和特殊效果
- ⏱‍多种游戏模式
- Ἰ9 成就系统和排行档
- 🔊 浓郁的声音效果

## 开始游戏

```bash
npm install
npm start
```

## 游戏规则

1. 点击任意两个水果进行交换
2. 当3个或更多相同水果连接时会自动消除
3. 消除后水果会下落，上面会补充新的水果
4. 连续消除可以获得更高分数
5. 在时间内达到目标分数即可胜利

## 特殊效果

- ☀️ 万能快：消除周围所有水果
- 🔥 小火炮：消除直线上的水果
- ⏰ 钟面：随机消除水果
